PARA USARLO:
1. Descomprime el ZIP.
2. Abre index.html con Chrome o Edge.
3. Captura nombre, grupo y matrícula.
4. El alumno lee el PDF y responde el reactivo de abajo.
5. Para avanzar debe seleccionar respuesta.
6. Al terminar, guarda resultado y exporta Excel.

PARA INGRESAR DESDE CUALQUIER DISPOSITIVO:
Sube la carpeta a GitHub Pages, Netlify o Firebase Hosting.
Después copia el enlace público y úsalo en el botón "Código QR".
